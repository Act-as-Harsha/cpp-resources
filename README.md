# 📚 C++ Resources

A collection of **C++ learning materials and reference books** for beginners and intermediate programmers.
This repository contains useful **PDF resources and books** to help understand C++ concepts, modern C++ practices, and programming techniques.

---

## 📂 Repository Structure

```
cpp-resources/
 └── Cpp-materials/
      ├── Effective Modern C++.pdf
      ├── A Tour of C++.pdf
      └── Other C++ reference materials
```

---

## 📘 Included Books

### 1️⃣ Effective Modern C++

Author: **Scott Meyers**

Topics covered:

* Smart pointers
* Move semantics
* Lambda expressions
* Modern C++ best practices

---

### 2️⃣ A Tour of C++

Author: **Bjarne Stroustrup (Creator of C++)**

Topics covered:

* C++ basics
* Object-oriented programming
* Generic programming
* Standard library overview

## 🎯 Purpose of This Repository

This repository is created to:

* 📖 Provide **learning resources for C++**
* 💻 Help students **understand modern C++ concepts**
* 🚀 Support programmers in improving **C++ coding skills**

---

## 🧠 Recommended Learning Path

1. Start with **A Tour of C++**
2. Practice basic C++ programs
3. Move to **Effective Modern C++** to learn advanced practices
